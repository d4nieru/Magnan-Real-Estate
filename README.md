## Features

Features include:
* Post an ad
* View all properties for rent
* View all properties for sale
* View all apartments
* Show all houses
* Show all villas
* Display all the parking lots
* A contact page to contact the owner of the site
* An Admin page for deleting and editing ads posted by users.
